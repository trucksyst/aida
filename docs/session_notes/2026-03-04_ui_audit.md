### 13. Detail Panel — Notes layout fix
**Файл:** `ui/app.js` (строка ~761)
- Notes теперь рендерится через `detail-row--block` (label сверху, текст снизу)
- Если notes пустое — строка не показывается вовсе
- Было: `detailRow('Notes', ...)` → слипание "NotesRGN or Lowboy..."
- Стало: `<div class="detail-row detail-row--block">` — вертикальный layout

### 14. Detail Panel — Rate display fix
**Файл:** `ui/app.js` (строки ~742–749)
- Когда rate = `—` (нет цены), блок `detail-rate-big` полностью скрывается
- Раньше показывался зелёный дефис, выглядевший как полоска
- `style="margin-top:4px"` → CSS-класс `.detail-rate-wrap`
